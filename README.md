QuizNova

QuizNova is an AI Powered Quiz Generator web application developed using HTML, CSS and JavaScript.

The application generates quiz questions dynamically using the Gemini API based on the topic and difficulty selected by the user.  
It also provides timer based quiz solving, answer explanations, score tracking and performance review.

# Features

- AI Generated Quiz Questions
- Topic Based Quiz Creation
- Easy / Medium / Hard Difficulty Levels
- Dynamic Question Loading
- Countdown Timer
- Instant Answer Checking
- Answer Explanations
- Score Calculation
- Best Streak Tracking
- Result Review Section
- Responsive User Interface
- Offline Backup Questions Support

# Technologies Used

# How To Run The Project
 1. Clone Repository
bash
git clone https://github.com/PearkyPearl/QuizNova.git

2. Open Project Folder
bash
cd QuizNova

 3. Run Project

Open `index.html` in your browser.

# Gemini API Configuration

Open `syn.js`

Find:
javascript
const API_KEY = "";

Replace with your Gemini API key:
javascript
const API_KEY = "YOUR_API_KEY";
